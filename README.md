
# Healthcare Claims Dashboard (Power BI)

## Project Overview

This project presents a healthcare claims analytics dashboard built in Power BI to monitor healthcare utilization, identify high-cost drivers, and support data-driven healthcare management decisions.

The dashboard integrates medical, dental, and pharmacy claims into a centralized reporting solution that enables stakeholders to track trends, analyze costs, and identify high-risk populations.

---

## Business Problem

Healthcare stakeholders required a unified reporting solution to:

- Monitor claims volume across multiple service types
- Identify high-cost medications and treatments
- Track utilization trends over time
- Detect high-risk populations
- Improve cost management and care prioritization

<!--Existing reporting methods were fragmented and time-consuming, limiting actionable insights.-->

---

## Dataset Information

- **500 enrolled members**
- **~8,000 total claims**
- **5 structured data tables**
- **Claim types included:**
  - Medical
  - Dental
  - Pharmacy

### Key Fields Included

- Member ID
- Claim Type
- Service Date
- Allowed Cost
- Provider Category
- Drug Name (Pharmacy)
- Diagnosis Category

## Dashboard Features

The dashboard includes:

- KPI summary metrics- Medical, Dental Pharmacy
- Claims trend analysis

- Member utilization insights
- Page Navigation, interactive filtering and bookmarks 

---

## Key Metrics

- **~8,000 total claims analyzed**
- **~$6M total healthcare cost**
- **0.7% decrease in total claims**
- **3% reduction in overall plan cost**

---

## Key Insights

- Pharmacy claims contributed significantly to total plan cost.
- A small percentage of members accounted for a disproportionately high share of total costs.
- Certain medications were identified as major cost drivers.
- Claims trends helped identify emerging high-risk populations.

---

## Business Impact

This dashboard enables stakeholders to:

- Identify high-cost medications
- Prioritize high-risk members
- Improve cost visibility across claim categories
- Support targeted care management strategies
- Optimize healthcare spending decisions
